INTRODUCTION OF JAVASCRIPT & WHAT IS JAVASCRIPT:

JavaScript was invented by Brendan Eich in 1995
JavaScript is a programming language used to make web pages interactive and dynamic. It runs in the browser and is used to give instructions to the computer, especially for handling user actions like clicks, form submissions, animations, etc.
